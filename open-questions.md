## What is the project actually called?

Two names are in use and they don't match:

- **SportsOnTheInternet** — the brand name per `brand.md`, used in page titles, headers, and copy. Intended domain `sportsontheinternet.com` is listed as "TBC — not yet registered."
- **sportsontheworldwideweb** — the actual GitHub org name, and the name baked into both real repos (`sportsontheworldwidewebadmin`, `sportsontheworldwideweb.github.io`) and the live published URL (`https://sportsontheworldwideweb.github.io/`).

Needs a decision: pick one name and bring the other into line — brand copy, repo names, and (if it ever happens) domain registration. Renaming GitHub repos affects the live URL and remotes, so treat that part as a deliberate, confirmed action when the time comes, not a quick find-and-replace.
